BIG DATA CLASS PROJECT

PROBLEM STATEMENT : MINI SQL using Map Reduce

This is achieved using the following three files:
one.py-for the user to give sql commands and call map reduce jobs
mapper.py
reducer.py

Execution: python3 one.py

Commands supported:
LOAD database_name/table_name AS (column_name:data_type,column_name:data_type,..)
SELECT column_name FROM database_name/table_name WHERE column_name=value AGGREGATE_BY count(or avg or sum)
